# Project Dynamo

A shmup built with PixiJS.

## Demo

Play it here: https://kitanga.github.io/project-dynamo/dist/

## Design spec

[Design.md](./Design.md)

## Controls

- `WASD`/`Arrow Keys` for movement
- `n`/`Spacebar` for shooting
